# Stock Portfolio Management System (Real-Time Price Fetching)

A console-based Stock Portfolio Management System in C that allows users to manage their stock investments and fetch near real-time stock prices using Yahoo Finance API. This project is built as a college project with emphasis on clean design, file handling, and practical financial use-cases.

## Features

- Add, update, and delete stocks in your portfolio  
- Store stock symbol, quantity, and buy price  
- Fetch live market price using Yahoo Finance API (via libcurl)  
- Automatic profit/loss calculation per stock  
- Total portfolio value and overall profit/loss  
- File-based data storage using .dat files  
- Includes terminal-based ASCII graph visualization for real-time stock price trends without using any GUI libraries  
- Simple, fast, menu-driven console interface  

## Tech Stack

- **Language:** C  
- **API:** Yahoo Finance (unofficial API endpoints)  
- **Library:** libcurl  
- **Compiler:** GCC
